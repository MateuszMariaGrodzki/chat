## Run client app

### Requirements

- `Node.js` **&ge;10.20.1**
- `npm` **&ge;6.14.4**

### Dev mode

1. Go to `/app/` directory
2. Run command:

```
npm i
```

3. After dependencies are installed, run:

```
npm run start
```

4. Enjoy application running on port 8000!
